    Custodian v0.0.1 API Reference

    Modules

    Custodian

    Custodian is a GitHub bot to manage routine development tasks. At a high-level it attempts to do the following

    Custodian.Bots

    The Bots context provides a boundary into the Custodian.Bots.Bot schema. It provides functions for listing, creating, updating, and deleting bots

    Custodian.Bots.Bot

    A Bot is installed on an individual repo on GitHub. It holds data about its repo and installation

    Custodian.Github

    The GitHub context provides a boundary into the GitHub API client interface and associated domain logic. The top-level class provides a way to process webhook payloads from GitHub

    Custodian.Github.Client

    Behaviour interface for taking actions on specific repos on GitHub

    Custodian.Github.Labels

    Behaviour interface for interacting with labels on pull requests on GitHub

    Custodian.Github.Merges

    Behaviour interface for merging branches on GitHub

    Custodian.Github.Processor

    Processing functions for each type of event from GitHub’s webhook that we respond to

    Custodian.Github.References

    Behaviour interface for interacting with branches on GitHub

    Custodian.Github.Tentacat.Client

    Provides an implementation of the Custodian.Github.Client behaviour with the GitHub API

    Custodian.Github.Tentacat.Labels

    Provides an implementation of the Custodian.Github.Labels behaviour with the GitHub API

    Custodian.Github.Tentacat.Merges

    Provides an implementation of the Custodian.Github.Merges behaviour with the GitHub API

    Custodian.Github.Tentacat.References

    Provides an implementation of the Custodian.Github.References behaviour with the GitHub API

    Custodian.Repo
    Custodian.Tasks

    Behaviour interface for processing tasks

    Custodian.Tasks.Async

    Provides an implmenetation of the Custodian.Tasks behaviour

    Custodian.Tasks.Sync

    Provides an implmenetation of the Custodian.Tasks behaviour

    CustodianWeb

    The entrypoint for defining your web interface, such as controllers, views, channels and so on

    CustodianWeb.Endpoint
    CustodianWeb.ErrorView
    CustodianWeb.FallbackController

    Translates controller action results into valid Plug.Conn responses

    CustodianWeb.Router
    CustodianWeb.Router.Helpers

    Module with named helpers generated from CustodianWeb.Router

    CustodianWeb.WebhookController

    Responds to the GitHub webhook and sends the payload for processing
